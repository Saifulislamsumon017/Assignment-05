Question-2: What is the difference between innerHTML, innerText, and textContent ?

There are difference between innerHTML, innerText, and textContent:
1.innerHTML: Returns or sets the HTML content inside an element, including HTML tags.
2.innerText: Returns or sets the visible text inside an element, ignoring hidden elements and preserving spacing.
3.textContent: Returns or sets all text inside an element, including hidden elements, without parsing HTML.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where a parent element listens for events on its child elements using event bubbling. Instead of adding event listeners to multiple child elements, a single event listener is added to a common ancestor, improving performance and efficiency.

Question-4: What is event bubbling in the DOM?

Event bubbling is a DOM event propagation mechanism where an event starts from the target element and moves up to its ancestors (from child to parent). This allows parent elements to handle events triggered by their child elements.
